# Hepsiburada test automation

Cucumber-Java Skeleton kullanılarak hepsiburada test caseleri yazılmıştır.

Çalıştırmak için

Maven:

    mvn test

yada ide yardımı ile çalıştırabilirsiniz.

Versions

```
java 1.8
junit 4.13.2
cucumber 6.10.4
maven compiler 3.8.1
maven surefire 2.22.2
selenium 3.141.5
```
